### Creating venv

python3 -m venv dattabot_venv

### Activating venv

Windows

`dattabot_venv\Scripts\activate.bat`

Windows - powershell

`dattabot_venv\Scripts\Activate.ps1`

Linux/Mac

`source dattabot_venv/bin/activate`

### Deactivating venv

deactivate

### Pip freeze

`pip freeze > requirements.txt`
